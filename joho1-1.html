演習問題　― コンピュータのしくみ ―

各問題に答えて「答え合わせ」ボタンを押してください。わからないときは「ヒント」を活用しましょう。

問題 1 組み合わせ選択
コンピュータの5大装置と動作サイクルについて、次の空欄（ア）〜（オ）に当てはまる語句の組み合わせとして正しいものを①〜④から１つ選べ。
  • CPU は（ア）と（イ）をまとめた呼び名である。
  • （ア）はプログラムの命令を読み出し、各装置に指示を出す司令塔である。
  • （イ）は四則演算や論理演算を実際に行う。英語名は（ウ）である。
  • CPU がメモリから命令を読み出すステップを（エ）、演算結果をメモリに書き戻すステップを（オ）という。
（ア）（イ）（ウ）（エ）（オ）
①制御装置演算装置ALUフェッチライトバック
②演算装置制御装置CUデコードフェッチ
③制御装置演算装置CUフェッチデコード
④演算装置制御装置ALUライトバックデコード
💡 ヒント：基礎知識

● CPU（Central Processing Unit）は2つの装置で構成されています。

● 「司令塔」＝ 命令を読み出して各装置に指示する装置は、英語で Control Unit（CU）といいます。

● 「計算エンジン」＝ 実際に演算を行う装置は、英語で Arithmetic and Logic Unit（ALU）といいます。

● 動作サイクルの順番：「取ってくる → 解読する → 計算する → 書き戻す」

● fetch（フェッチ）＝ 取ってくる　／　write-back（ライトバック）＝ 書き戻す

📝 解答・解説

正解：①

（ア）制御装置＝CU　（イ）演算装置＝ALU　（ウ）ALU　（エ）フェッチ　（オ）ライトバック

動作サイクルは フェッチ → デコード → 実行 → ライトバック の順です。

各語句の意味

制御装置（CU：Control Unit）　… プログラムの命令を順番に読み出し、各装置に「次はこれをしなさい」と指示を出す司令塔。

演算装置（ALU：Arithmetic and Logic Unit）　… 四則演算や論理演算を実際に行うCPUの計算エンジン。

フェッチ（fetch）　… メモリから命令を取ってくる最初のステップ。本棚から本を手に取るイメージ。

ライトバック（write-back）　… 演算結果をメモリに書き戻す最後のステップ。答えをノートに書き留めるイメージ。

問題 2 正誤の組み合わせ
2進数・フリップフロップ・RAMに関する次の記述ア〜エについて、正しいものの組み合わせを①〜④から１つ選べ。
ア 10進数の「10」を2進数で表すと「1010」である。
イ フリップフロップは1個で1バイト（8ビット）を記憶する最小単位である。
ウ RAMは Random Access Memory の略で、電源を切ってもデータは消えない。
エ AND演算は、2つの入力がどちらも1のときだけ出力が1になる。
正しい記述の組み合わせを選べ。
💡 ヒント：基礎知識

● ア：10 ＝ 8＋2 ＝ 1×2³＋0×2²＋1×2¹＋0×2⁰ → 各桁を並べると？

● イ：フリップフロップ1個は「0か1の1桁分」だけ記憶できます。1バイト＝8ビットには何個必要でしょうか？

● ウ：RAMは「作業台」のたとえで学びました。電源を切るとどうなりますか？　→ 揮発性・不揮発性を思い出して。

● エ：AND＝「かつ」。両方が1のときだけ出力1。1 AND 0 → ? 　0 AND 0 → ?

📝 解答・解説

正解：②（アとエ）

ア 正しい。 10 ＝ 8＋2 ＝ 1×2³＋0×2²＋1×2¹＋0×2⁰ ＝ 1010（2）

イ 誤り。 フリップフロップ1個は 1ビット（0か1）を記憶する最小単位です。1バイト＝8ビットには8個必要です。

ウ 誤り。 RAMは揮発性で、電源を切るとデータは消えます。

エ 正しい。 AND演算は両入力が1のときのみ出力1。1 AND 0→0、0 AND 0→0。

【補足】なぜ揮発性・不揮発性なのか

DRAM：コンデンサに電荷を蓄えて記憶。常にリフレッシュが必要で、電源が切れると消えます。D＝Dynamic（動的）の略。　例：パソコンのメモリ棒（DDR5）、スマホのメモリ（LPDDR5）

SRAM：フリップフロップ回路で記憶。電源停止で消えます。S＝Static（静的）の略。　例：CPUのL1/L2/L3キャッシュメモリ

SSD・フラッシュメモリ：絶縁体で電子を閉じ込めて記憶。電源なしでも保持されます。　例：USBメモリ・SDカード・スマホのストレージ

💡 たとえ：RAM（揮発性）は「作業台」、SSD（不揮発性）は「棚」。Ctrl+S で保存するのは、作業台のものを棚に移す操作です。
