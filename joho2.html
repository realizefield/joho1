コンピュータの構成 (1)
ハードウェア・ソフトウェア・5大装置

コンピュータは大きく「ハードウェア」と「ソフトウェア」の2つの要素からできています。 その中核となるCPUを含む5大装置の構成は共通テストで頻繁に出題されます。しっかり押さえましょう。

1コンピュータの基本構成

コンピュータは大きく「【ハードウェア】」と「【ソフトウェア】」の2つの要素からできています。

【ハードウェア】 【ソフトウェア】
物理的な装置・部品のこと
CPU・メモリ・キーボード・ディスプレイ・HDDなど
コンピュータを動かす仕組み・プログラム
OS（基本ソフトウェア）・アプリ（応用ソフトウェア）など

2コンピュータの【5大装置】

コンピュータのハードウェアは、次の【5大装置】から構成されています。これは共通テストで頻繁に出題されます。

装置名英語名役割・説明
【演算装置】 四則演算やデータの比較・判断を行う。CPUの中にある。
【制御装置】 プログラムから命令を取り出し、解釈して各装置に指示を出す。CPUの中にある。
【記憶装置】 Memory プログラムやデータを記憶する。主記憶（RAM）と補助記憶（HDD/SSD）がある。
【入力装置】 Input プログラムやデータを外部から取り込む。キーボード・マウス・スキャナなど。
【出力装置】 Output 処理されたデータを外部に出力する。ディスプレイ・プリンタ・スピーカーなど。
